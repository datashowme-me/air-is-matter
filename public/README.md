# Donation QR Code Image

Place your donation QR code image in this folder with the filename: **donation-qr.png**

Supported formats:
- PNG (recommended)
- JPG/JPEG
- SVG
- WebP

The image should be square (recommended: 512x512px or larger) for best display in the modal.

If you use a different filename, update the image path in:
`/components/DonationModal.tsx` (line 43)
